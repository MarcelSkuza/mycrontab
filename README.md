# Mycrontab

Bash script for crontab file editing/management compatible with most Linux distributions.

## Usage

**Interaction:**\
1 - Display crontab jobs\
2 - Insert a job\
3 - Edit a job\
4 - Remove a job\
5 - Remove all jobs\
9 - Exit
